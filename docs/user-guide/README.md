# 👥 User Guide

Podręczniki użytkownika dla systemu Gościniec Rodzinny.

## 📋 Zawartość

- `RESERVATIONS_USER_GUIDE.md` - Podręcznik obsługi modułu Rezerwacje
- `QUEUE_USER_GUIDE.md` - Jak korzystać z systemu kolejki
- `DEPOSITS_USER_GUIDE.md` - Zarządzanie zaliczkami
- `FAQ.md` - Najczęściej zadawane pytania

## 🎯 Dla kogo?

Ta dokumentacja jest przeznaczona dla:
- Pracowników restauracji
- Menedżerów
- Administratorów systemu

## 🚀 Szybki start

1. Zaloguj się do systemu
2. Przejdź do sekcji **Rezerwacje**
3. Kliknij **Nowa Rezerwacja**
4. Wypełnij formularz
5. Kliknij **Zapisz**

## 📞 Pomoc

W razie pytań skontaktuj się z administratorem systemu.
